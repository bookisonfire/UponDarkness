# Upon Darkness

Upon Darkness is a terminal-based text adventure game where you explore a medieval dungeon filled with treasures, mysteries, and danger. Make choices wisely to survive, uncover secrets, and escape the dungeon alive!

## Features
- Interactive text-based adventure in your terminal/command prompt
- Typewriter-style text effect to enhance immersion
- Multiple scenes and story branches
- Save and load your progress at any time
- Track your decisions throughout the game

### Requirements
- Python 3.7 or higher
- Compatible with Windows, macOS, and Linux

### How to Download
Clone or download this repository:
`git clone <repository-url>`
Or download the ZIP file and extract it to a folder.

### How to Run the Game
1. Open your terminal or command prompt
2. Navigate to the folder containing UponDarkness.py:
`cd path/to/game/folder`
3. Run the game using Python:
`python UponDarkness.py`
> NOTE: On some systems, you may need to use 'python3' instead of 'python'
4. Follow the on-screen instructions

### Saving and Loading
The game automatically saves your progress to game_data.json when you choose [s] Save Game.
When you open the game, your system will search for a save file. If it finds one, it will give you the option to load it, or start a new game.
